Lab 7: Functions

The focus of this lab was to write functions that take user input and manipulate a string.

Challenges

With a little research and debugging, I was able to get my functions to do what I wanted. Since I had some JavaScript knowledge going into this, I chose to use ES6 arrow functions.

Problems

The functions themselves were OK. I had some problems working on some bonus tasks, such as converting a string to title case. Again, with some research and debugging, I was able to get it to work. Real talk, the biggest problem for this lab was CSS. Specifically, trying to position the shuffled name within the name tag image. I got it to look like I wanted, but once you resize the browser window, all bets are off.

Results

This was a challenging lab, particularly the bonus tasks. In the end, I like how things turned out.
