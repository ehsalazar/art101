Lab 11: Libraries & jQuery

The focus of this lab was to experiment with jQuery.

Challenge

The challenge for this lab was to create buttons that when clicked, would alter the styling of the section they were in. This needed to be done using jQuery, rather than vanilla JavaScript. Overall, this lab wasn’t overly complicated.

Problems

Once I discovered the usefulness of the $('element').toggleClass('class_name'); method in jQuery, there were no problems. I was able to make use of the method for both the main lab tasks and for the bonus task of styling the buttons themselves.

Results

I was able to find the elements I wanted, add new elements, and alter class selectors all with jQuery. In the end, it was a simpler and cleaner approach to doing tasks we had previously done using vanilla JavaScript.
