Lab 13: Loops

The focus of this lab was to experiment with loops.

Challenge

In this lab we explored looping by working through FizzBuzz, a common programming challenge. We need to loop through a set of numbers, in this case 1–200, and then output a particular word if the number was evenly divisible by another particular number. For example, if one of the numbers we looped through was divisible by 3, would we output “Fizz.” We then had to figure out how to get our output from the function into our HTML.

Problems

Having written FizzBuzz functions in the past, that aspect of this lab wasn’t overly difficult. The problem I encountered was in trying to get the output into my HTML formatted in a specific way. One approach would be to construct one long string that gets concatenated in each iteration of the loop. In the end, I didn’t like the lack of control or customization that allowed. Instead, I appended a new HTML element as part of each loop iteration.

Results

With some trial, error, and learning, I was eventually able to get the full set of 1–200 formatted and into my HTML, as seen here.

Fizz = evenly divisible by 3, Buzz = evenly divisible by 5, Boom = evenly divisible by 7
