Lab 12: Conditionals

The focus of this lab was to create a function that depends on conditionals.

Challenge

This lab focused on our ability to incorporate conditionals into our JavaScript functions. We did this by taking an inputted string and then used its length, the modulus operator, and the number of Hogwart’s houses to determine the division remainder. We then used a series of if, else if, and else conditional statements to match the remainder to one of the four house.

Problems

The main lab didn’t present any major problems. However, attempting to complete the lab’s bonus tasks of using an array of objects and a hash function required extra research and attempts.

Results

With the code for the initial tasks refactored, and thanks to the executed bonus tasks, I was able to successfully return not only the house the user gets sorted into, but also a description of that house. Both stages of the lab also successfully used jQuery.
