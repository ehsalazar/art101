Lab 16: JSON & APIs

The focus of this lab was to experiment with processing JSON from an API.

Challenge

Building off of the previous lab, we worked in this lab to not only make successful AJAX functions, but to also successfully navigate and use more complex JSON objects that come as responses. Using XKCD’s API, we sought to retrieve the most current comic and then render it within our HTML webpage. If successful, we could then attempt to retrieve other comics and provide a way for our users to explore them.

Problems

Getting the most current comic was relatively simple. Thankfully, the returned JSON object wasn’t overly complex and provided a number of key–value pairs which aided in our output functions. The bigger challenge came from the lab’s bonus tasks. Each subsequent AJAX call would depend upon the JSON object returned from the first call. It took some trial and error, as well as debugging, to ensure that the key–value pairs we thought we were working with, were in fact what we needed.

Results

With all said and done, I was able to successfully retrieve the most current XKCD comic and then offer the user options to retrieve other comics. A series of subsequent AJAX calls now also provide the very first coming, a previous comic, a random comic, the next comic, and finally the most current comic.
