Lab 10: JavaScript Events and Forms

The focus of this lab was to experiment with JS events.

Challenge

Building off Lab 7 made this lab more manageable. Thanks to practice in class and past bonus tasks, I was able to find and create elements using JavaScript. As I had in Lab 7, I wanted the output of my function to render in the image of the name tag. While I used an already created HTML element for my general output, I wanted to create a new element for the returned, scrambled name.

Problems

With the main tasks of the lab completed, I wanted to give the bonus tasks a go. I wasn’t sure how I was going to be able to glitch the entire screen and then go back to the regular lab page. Thanks to some research, I found and ultimately used element.style.display = "none"; and element.style.display = "block"; methods. If I had time to go farther, I might try to get the screen to glitch only on the first click of the button, rather than every time.

Results

The form field accepts user input and upon clicking the “submit” button, the webpage glitches and then returns to show an anagram of the user’s name.
