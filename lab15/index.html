Lab 15: AJAX

The focus of this lab was to experiment with jQuery and AJAX.

Challenge

Through experimenting, we started to understand the concept of API calls, how to make them, and what to do with the data that’s returned. We’re using a button click event to trigger it all into action.

Problems

After trying a few APIs, I found that the returned data was often in different forms. Some returned simple arrays, while others returned more complex JSON objects. The API I decided to use was somewhere in the middle. The returned JSON was simple enough, or perhaps not as nested as others, which allowed me to access the specific portion of the object I wanted. In this case, an image or video url. With that pinpointed, I then had to figure out a way to create the appropriate HTML tag, which I would use to get the output into my webpage.

Results

After a few attempts and thanks to debugging in the console, I was eventually able to get the API calls executed successfully and the returned data outputted as desired.
