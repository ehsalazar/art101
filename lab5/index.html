Lab 5: Data Types and Variables

The focus of this lab was to experiment with basic JavaScript data types and variables.

Challenges

Using Chrome’s browser console was straightforward, as was declaring and initializing variables in lab.js. The challenge came when trying to output the JavaScript to HTML. After a couple of initial failures, I was able to get document.write() to work. Eventually, that lead to success with document.writeln().

Problems

I chose to use string interpolation for the variables we created in this lab. I had used that method before, instead of traditional string concatenation. It took a couple of attempts to recall the precise formatting needed to make it work.
