Lab 9: JavaScript for the Web

The focus of this lab was to experiment with DOM manipulation.

Challenges

Similar to the last lab, this one wasn’t as hard. It made even more sense, having attempted the previous bonus tasks.

Problems

With the main tasks out of the way, I moved on to the bonus tasks for this lab. I was able to find the insertBefore() method, which allowed me to add a new element to the top of an existing element. I also found the prepend() method, which is the one I ended up going with. I wasn’t sure if it would work with vanilla JavaScript, as I had mostly seen it in relation to jQuery. It did work in the end, although I’m not completely sure why.

Results

I was able to successfully create new elements in JavaScript and add them to my HTML. Here you can see three new <p> elements that were either appended or prepended to an existing <div> element. The styling was also added in JavaScript, although understandably that’s not the best solution for styling.
